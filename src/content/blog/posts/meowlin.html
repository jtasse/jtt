May 13, 2026 | James Tasse

That bird app

You may have heard of Merlin, a fantastic mobile app that can identify birds in real time.

This cat app

Well I had a very simple idea. What if I created a solution like Merlin but for cats? Thus Meowlin was born.

NOTE: you can check out the demo at https://jtasse.github.io/meowlin/

AWS + React Demo

All right, before we go any further, I need to clarify that one cannot simply walk into a cat cafe and use Meowlin to start identifying cat breeds based on recorded meows. This is first and foremost a demo of the knowledge I gained before passing the AWS Certified Developer - Associate exam; along with an extension of my React experience.

Caveats

As such, I list a few solution caveats in the solution's README.

But the big one is worth pointing out right now: I implemented mock processing for this demo rather than gathering loads of high quality meow audio and using machine learning to train an audio processing model. Phew!

But who knows, maybe someday!
