## About This Demo

The ORC demo is an interactive 3D experience built with Three.js. It includes:

- Satellite selection and tracking
- Robotic hand actuator animations
- Simulated decommissioning procedures
- A smart camera system

Visit the [ORC Demo](/portfolio/orc-demo) to interact with the system.
